# Towry's hand crafted NeoVim config

Works without plugins.

## Commands

### `PrebundlePlugins`

will bundle all files inside `user.plugins` to a single file `user.plugins_bundle`

## FYI

- If there is something wrong with treesitter, try run `TSUpdate`.
